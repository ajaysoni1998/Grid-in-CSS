In this example, we have a grid container with two columns. Width of 1st column set to 200px and height of 1st row set to 100px with grid-auto-rows set to 50px and grid-auto-columns set to 100px. This means that any additional rows created beyond the first one will have a height of 50 pixels and any additional columns created beyond the first one will have a width of 100 pixels. The grid items are placed in this grid, and you'll notice that items 3 to 6 create additional rows with a height of 50px.
